Background

What is this project?

This project builds a gesture-controlled 3D CAD viewer inspired by the holographic interfaces seen in science fiction. Instead of a mouse or controller, the system uses your webcam to detect and track the skeleton of your hand in real time, mapping natural hand movements to 3D camera controls.

The web prototype is fully functional — load any .GLB, .GLTF, or .OBJ file, hold up your hand, and rotate your CAD model by moving your palm. The implementation uses MediaPipe Hands (21-joint skeleton) and Three.js with full PBR rendering including Image-Based Lighting so metallic surfaces show their real colours.

The project is now being ported to Unity 6 + Meta Quest 3, where the same gesture logic runs in full 6-DoF virtual reality — letting you physically hold and turn a virtual model as if it were in your hands.

VR Hand Tracking
Unity's XRHandSubsystem provides 26 joints per hand in 3D world space. The same Rotate / Zoom / Pan gesture logic from the web prototype maps directly to wrist and fingertip poses.

Model Format
The project uses the GLB binary format which embeds all PBR textures (base colour, metalness, roughness, normal maps) in a single file. The bus model contains 43 meshes and 44 texture maps.

Interaction Design

Three Gesture Modes

A rule-based classifier reads joint positions each frame and maps hand poses to 3D camera controls. Only one gesture is active at a time.

✋
Rotate
Open fist — palm facing camera
Move your open hand left/right to rotate the model around the Y axis. Move up/down for X axis. The left-right axis is mirrored so the model always turns in the same direction as your hand — natural and intuitive.
🤌
Zoom
Pinch — thumb + index tip < 2.5cm
Bring your thumb and index fingertip together. Move your hand toward your body to zoom in, away to zoom out. The pinch distance ratio relative to hand size keeps detection consistent across different hand sizes.
✌️
Pan
Two fingers — index + middle extended
Extend your index and middle fingers (like a peace sign) while keeping ring and pinky folded. Move your hand to translate the model in the viewport plane without rotating it — useful for repositioning after zooming in.
Processing Pipeline
Step 01 📷
Webcam Capture
MediaDevices API streams 30fps video to <video> element
Step 02 🧠
ML Detection
MediaPipe model outputs 21 normalised (x,y,z) joint coords
Step 03 🖐
Gesture Classify
Pinch ratio + finger extension → Rotate / Zoom / Pan
Step 04 ⚙️
State Update
Palm delta updates target rotation, zoom, or pan offset
Step 05 〰️
Smooth Lerp
Linear interpolation (factor 0.08) damps camera movement
Step 06 🖥
3D Render
Three.js draws scene with IBL, shadows, PBR materials

// Classifies the current hand pose from 21 MediaPipe landmarks
function detectGesture(lm) {
  // Pinch: ratio of thumb-index distance to hand size
  const pinchRatio = ldist(lm[4], lm[8]) / ldist(lm[0], lm[5]);

  // Finger extension: tip Y above MCP Y means finger is up
  const indexUp  = lm[8].y  < lm[6].y;
  const middleUp = lm[12].y < lm[10].y;
  const ringUp   = lm[16].y < lm[14].y;
  const pinkyUp  = lm[20].y < lm[18].y;

  if (pinchRatio < 0.25)                    return 'zoom';
  if (indexUp && middleUp && !ringUp && !pinkyUp) return 'pan';
  return 'rotate';    // open fist = default
}

// Apply gesture to scene state — called every frame
function applyGesture(lm, gesture, palmX, palmY) {
  if (gesture === 'rotate' && hand.prevPalm) {
    const dx = palmX - hand.prevPalm.x;
    const dy = palmY - hand.prevPalm.y;
    state.targetRotY -= dx * 4.0;  // negated: model follows hand direction
    state.targetRotX += dy * 4.0;
  }
}

function buildEnvScene() {
  const envScene = new THREE.Scene();
  const skyMat  = new THREE.MeshBasicMaterial({
    color: 0xdde4ec, side: THREE.BackSide
  });
  envScene.add(new THREE.Mesh(
    new THREE.SphereGeometry(50), skyMat
  ));
  // Key light panel, fill, rim, top softbox ...
  return envScene;
}

// Generate PMREM map and apply as scene IBL
const envTexture = pmremGen.fromScene(
  buildEnvScene(), 0.04

C# · Unity XR Hands
void Update() {
  if (hs == null) return;
  XRHand hand = hs.rightHand;
  if (!hand.isTracked) { wasTracked = false; return; }

  // Get joint world-space poses
  hand.GetJoint(XRHandJointID.Wrist)    .TryGetPose(out var wrist);
  hand.GetJoint(XRHandJointID.IndexTip) .TryGetPose(out var idx);
  hand.GetJoint(XRHandJointID.ThumbTip) .TryGetPose(out var thumb);
  hand.GetJoint(XRHandJointID.MiddleTip).TryGetPose(out var mid);

  float  pinch  = Vector3.Distance(thumb.position, idx.position);
  bool   idxUp  = idx.position.y > wrist.position.y + 0.06f;
  bool   midUp  = mid.position.y > wrist.position.y + 0.06f;
  bool   isPinch = pinch < 0.025f;
  bool   isPan   = !isPinch && idxUp && midUp;

  if (wasTracked) {
    Vector3 dp = wrist.position - prevPalm;
    if      (isPinch) { /* zoom via localScale */ }
    else if (isPan)   { transform.position += dp * 2f; }
    else {
      // Rotate — negate X to match intuitive direction
      transform.Rotate(Vector3.up,   -dp.x * 300f, Space.World);
      transform.Rotate(Vector3.right,  dp.y * 300f, Space.World);
    }
  }
  prevPalm = wrist.position; wasTracked = true;
}
JavaScript · Three.js
// Load GLB — preserves ALL embedded PBR materials (no colour override)
function loadGLTF(file) {
  return new Promise((resolve, reject) => {
    const loader = new THREE.GLTFLoader();
    const url   = URL.createObjectURL(file);
    loader.load(url,
      gltf => { URL.revokeObjectURL(url); resolve(gltf.scene); },
      null,
      err  => { URL.revokeObjectURL(url); reject(err); }
    );
  });
}

// After loading: scale to 2.5m bounding box, assign IBL env map
function fitModel(group) {
  const box    = new THREE.Box3().setFromObject(group);
  const size   = box.getSize(new THREE.Vector3());
  const scale  = 2.5 / Math.max(size.x, size.y, size.z);
  group.scale.setScalar(scale);

  group.traverse(child => {
    if (!child.isMesh) return;
    child.castShadow = child.receiveShadow = true;
    // Apply IBL — DO NOT change material colours
    [child.material].flat().forEach(m => {
      m.envMap          = envTexture;
      m.envMapIntensity = 1.0;
      m.needsUpdate     = true;
    });
  });
}
